# CSC 483 Algorithm Assignments

This workspace contains the implementation for the CSC 483 course assignments, covering search and sorting algorithms.

## Prerequisites

- Java Development Kit (JDK) 8 or higher installed on your system.

## Project Structure

- `com.csc483.assignment1.search`: Contains the implementation for Assignment 1 (Search Algorithms).
- `com.csc483.assignment2.sorting`: Contains the implementation for Assignment 2 (Sorting Algorithms).

## How to Compile and Run

Open a terminal or command prompt in the root directory of the workspace (`csc483-algorithm-assignment`).

### Assignment 1: Search Algorithms

1. **Compile the code:**
   ```bash
   javac -d . com.csc483.assignment1.search/Main.java com.csc483.assignment1.search/Product.java com.csc483.assignment1.search/ProductManager.java
   ```

2. **Run the application:**
   ```bash
   java com.csc483.assignment1.search.Main
   ```

### Assignment 2: Sorting Algorithms

1. **Compile the code:**
   ```bash
   javac -d . com.csc483.assignment2.sorting/Main.java
   ```

2. **Run the application:**
   ```bash
   java com.csc483.assignment2.sorting.Main
   ```

*(Note: Ensure that you are running the `java` commands from the root directory so that the package structure is resolved correctly and datasets can be accessed if they use relative paths).*
